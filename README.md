# Instalar

* Ctrl + Shift + p
* Digite: vsix
* Enter
* Selecione jopestpe-tema-0.0.2.vsix
* Instalar 

# Alterar

### Dependencias

```sh
npm install -g yo generator-code vsce
```

## Criar modelo do zero

```sh
yo code
npm init
```

## Empacotar e instalar no vsc (gerar .vsix)

```sh
vsce package
```
